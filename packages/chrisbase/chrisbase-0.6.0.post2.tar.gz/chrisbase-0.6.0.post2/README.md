# chrisbase

Base library for python coding

## Installation

* Install through pip
    ```bash
    pip install chrisbase
    ```

* Install through git
    ```bash
    rm -rf chrisbase*
    git clone git@github.com:chrisjihee/chrisbase.git
    pip install -U -e chrisbase*
    ```
